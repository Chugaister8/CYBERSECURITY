# Додаток A. Глосарій

## 3DES
Також відомий як Triple DES — блоковий шифр із симетричними ключами, що застосовує алгоритм DES тричі до кожного блоку.

## A

**Протокол AAA (AAA protocol)**
Автентифікація, авторизація та облік (Authentication, Authorization, and Accounting) — використовує різні протоколи канального рівня та автентифікацію за PAP або CHAP.

**Коди доступу (Access codes)**
Числова послідовність (зазвичай чотири цифри), що ідентифікує авторизованого клієнта або користувача. Після автентифікації коди доступу дозволяють поставити систему під охорону, зняти з охорони або скористатися іншими функціями.

**Контроль доступу (Access control)**
Заходи безпеки, що гарантують надання ресурсів лише тим користувачам, яким вони належать.

**Ворота контролю доступу (Access control gate)**
Прохід у стіні або паркані, що слугує переходом між двома відкритими просторами.

**Політика контролю доступу (Access control policy)**
Метод, за яким генерується список контролю доступу.

**Підтвердження (ACK)**
Частина другого і третього (фінального) кроку тристороннього рукостискання протоколу TCP/IP.

**Active Directory**
Служба каталогів Microsoft для управління доменом: автентифікація, авторизація, управління сертифікатами тощо.

**Активні RFID-теги (Active RFID tags)**
Теги з власним живленням для більшого радіусу дії.

**ActiveX**
Утиліта Microsoft для розширених інтерактивних функцій вебзастосунків. Завантажується автоматично, що створює потенційну вразливість.

**AFHSS (Adaptive Frequency Hopping Spread Spectrum)**
Адаптивне розширення спектру зі стрибками частоти. Реалізовано в Bluetooth у діапазоні 2,4 ГГц.

**Атаки ARP-спуфінгу (ARP spoofing attacks)**
Атака, де зловмисник надсилає підроблені ARP-повідомлення для прив'язки своєї MAC-адреси до IP-адреси іншого користувача.

**Адміністративні засоби (Administrative tools)**
Програми для управління пам'яттю, оптимізації жорсткого диска, конфігурування сервісів ОС та усунення проблем.

**Рекламне ПЗ (Adware)**
Небажана реклама, що відображається у веббраузерах.

**Алгоритм (Algorithm)**
У контексті безпеки — код для зміни повідомлення, щоб неавторизовані особи не могли його прочитати.

**Аналогова камера (Analog camera)**
Камера з окремим коаксіальним кабелем для підключення до монітора або пристрою запису.

**Android**
Операційна система для планшетів і смартфонів, найпопулярніша мобільна ОС.

**Сповіщувачі (Annunciators)**
Пристрої сигналізації, що сигналізують про стан системи за допомогою звуку, світла або іншого індикатора.

**Аналіз аномалій (Anomaly analysis)**
Застосування методів статистичного аналізу до потоку даних для визначення «нормальності» або «аномальності».

**IDS/IDPS на основі аномалій (Anomaly-based IDS/IDPS)**
Системи виявлення та запобігання вторгненням, що використовують аналіз аномалій.

**Анонімний проксі (Anonymous proxy)**
Проксі-сервер, що приховує оригінальну IP-адресу клієнта.

**Антишпигунське ПЗ (Antispyware)**
Програми для захисту від шпигунського ПЗ.

**Антивірус (Antivirus)**
Програми для захисту від комп'ютерних вірусів.

**Apple OS X**
Операційна система Apple на основі Unix.

**Апарат безпеки (Appliance-based virtualization)**
Закриті системи управління зберіганням між користувацькою та мережею зберігання.

**Пакети застосунків (Application packages)**
Розширення операційної системи для конкретних задач.

**Сервери застосунків (Application servers)**
Сервери для конкретної ролі в бізнес-діяльності організації.

**Актив (Asset)**
Щось цінне — матеріальне (флеш-накопичувач) або нематеріальне (дані на ньому).

**Асиметричні ключі (Asymmetric keys)**
Два ключі в алгоритмах шифрування: один для шифрування, інший для розшифрування.

**Аудит (Auditing)**
Форма обліку — заздалегідь запланований метод моніторингу для оцінки проблем.

**Автентифікація (Authentication)**
Процес підтвердження того, що особа є тією, ким вона стверджує бути.

**Сервер автентифікації (Authentication Server, AS)**
Сервер для надання мережевим користувачам автентифікації.

**Авторизація (Authorization)**
Ресурси, до яких користувач має дозвіл, та дії, які він може виконувати.

## B

**Носій резервної копії (Backup media)**
Носій для зберігання резервних копій даних.

**Політика резервного копіювання (Backup policy)**
Визначає, що, коли і де резервувати; хто відповідає за резервні копії.

**Пропускна здатність (Bandwidth)**
Загальна здатність носія переносити дані в певний момент.

**Бар'єри (Barriers)**
Перешкоджають просуванню зловмисника від зовнішнього периметра до внутрішньої зони.

**BIOS (Basic Input/Output System)**
Програма завантаження комп'ютера та управління операціями введення/виводу.

**Bastion host**
Розташований у DMZ — може бути міжмережевим екраном, маршрутизатором, сервером або групою комп'ютерів із прямим доступом до Інтернету.

**Чорний хакер (Black hat hacker)**
Особа з навичками програмування, що використовує їх для порушення або обходу мережевого захисту зі злочинними цілями.

**Чорний список (Blacklisting)**
Заборона доступу лише для користувачів, що відповідають певним критеріям.

**Блоковий шифр (Block cipher)**
Шифр, що застосовує алгоритм до блоку даних.

**Blowfish**
Блоковий алгоритм шифрування із симетричними ключами.

**Пастух ботів (Bot herder)**
Особа, що контролює ботнет.

**Ботнет (Botnet)**
Велика колекція зомбі-комп'ютерів, контрольованих пастухом ботів.

**Боти (Bots/Zombies)**
Комп'ютери під контролем неавторизованої особи.

**BYOD (Bring-Your-Own-Device)**
Дозвіл співробітникам використовувати власні пристрої для бізнес-діяльності.

**Широкомовний шторм (Broadcast storm)**
Широкомовний трафік, що передається кожним мережевим пристроєм, перевантажуючи мережу.

**Переповнення буфера (Buffer overflow)**
Аномалія, коли програма виходить за межі буфера — призводить до непередбачуваної поведінки, помилок пам'яті або збоїв.

**Атака переповнення буфера (Buffer overflow attack)**
Навмисне спричинення переповнення буфера системи.

**B2B (Business-to-Business)**
Мережеві канали для безпечних транзакцій між довіреними організаціями.

**B2C (Business-to-Customer)**
Спеціалізована DMZ для взаємодії організації з клієнтами.

## C

**Отруєння кешу (Cache poisoning)**
Атака, де зловмисник встановлює шахрайський DNS-сервер і використовує його для підміни записів основного DNS-сервера.

**CAPTCHA**
*Completely Automated Public Turing test to tell Computers and Humans Apart* — тест для розрізнення людей і автоматизованих інструментів.

**Складність (Complexity)**
Стосовно паролів — поєднання довжини, ширини та глибини.

**CIA (Confidentiality, Integrity, Availability)**
Конфіденційність, цілісність і доступність — класична модель інформаційної безпеки.

**Шифр (Cipher)**
В криптографії — алгоритм для шифрування даних.

**Замки-шифри (Cipher locks)**
Замки, що відмикають магнітний засув при введенні правильного коду.

**Зашифрований текст (Ciphertext)**
Текст будь-яких даних після шифрування криптографічним ключем.

**Відкрита текстова передача (Cleartext)**
Збережені або передані дані, що не зашифровані.

**Атака клікджекінгу (Clickjacking attack)**
Оманливі техніки фреймів для обманного кліку на вміст зловмисника.

**Клієнт-сервер (Client-server)**
Ряд користувачів із кількома робочими станціями, де дані зберігаються на сервері.

**Хмарні обчислення (Cloud computing)**
Використання віддалених мережевих серверів для зберігання та обробки даних.

**Коаксіальний кабель (Coaxial cable)**
Кабель передачі з двома провідниками в поліетиленовій оболонці.

**Cookie**
Невеликі файли, що вебсервери надсилають веббраузерам при відкритті їхніх сторінок.

**Отруєння cookie (Cookie poisoning)**
MITM-атака — зловмисник змінює вміст cookie та надсилає змінений cookie назад.

**Крадіжка cookie (Cookie theft)**
Отримання неавторизованого доступу до сайту через перехоплення cookie.

**CPE (Common Platform Enumeration)**
Стандартизований метод опису та ідентифікації ОС, апаратних пристроїв і класів застосунків.

**CVE (Common Vulnerabilities and Exposures)**
Система посилань на загальновідомі вразливості.

**CVSS (Common Vulnerability Scoring System)**
Система оцінки вразливостей CVE.

**Зломщики (Crackers)**
Особи, що зламують комп'ютерну безпеку для отримання доступу до ПЗ без покупки.

**CRAM-MD5**
Система автентифікації електронної пошти, що передає паролі в хешованому вигляді.

**Міжсайтовий сценарій (Cross-site scripting, XSS)**
Метод атаки, що дозволяє вставляти клієнтський сценарій у вебсторінку.

**Криптографія (Cryptography)**
Процедури, процеси та техніки перетворення даних у секретний код.

**Кіберзлочинці (Cybercriminals)**
Зловмисники, як правило, мотивовані жадібністю та прагнуть фінансової вигоди.

**Кібербезпека (Cybersecurity)**
Захист фізичного доступу до власності, систем та обладнання, а також електронного та інформаційного доступу до даних та засобів управління.

**Кібертерористи (Cyberterrorists)**
Особи, мотивовані просуванням політичної програми або ідеології.

## D

**Демон (Daemon)**
Стандартний обліковий запис із привілеєм виконувати фонові процеси без участі користувача.

**Шифрування диска (Disk-level encryption)**
Шифрування всієї структури диска.

**DAC (Discretionary Access Control)**
Дискреційний контроль доступу — користувач на власний розсуд визначає доступ до своїх об'єктів.

**DMZ (Demilitarized Zone)**
Окрема периметрична мережа, що ізолює захищену інтрамережу від зовнішнього світу, але надає публічний доступ до зовнішніх ресурсів.

**Відмова в обслуговуванні (Denial of Service, DoS)**
Атаки, спрямовані на перевантаження хоста, сервера або мережевого ресурсу до точки функціонального припинення надання послуг.

**DEP (Data Execution Prevention)**
Схема «пісочниці» для ізоляції програми в конкретній ділянці пам'яті.

**DHCP (Dynamic Host Configuration Protocol)**
Динамічне надання IP-адрес із пулу.

**DHCP snooping**
Технологія безпеки для фільтрації вхідних DHCP-повідомлень.

**Словникова атака (Dictionary attack)**
Систематична атака методом грубої сили, де кожне слово зі словника використовується як пароль.

**Цифрові сертифікати (Digital certificates)**
Цифрові верифікації того, що відправник зашифрованого повідомлення є тим, ким він стверджує бути.

**DNS (Domain Name System)**
Система доменних імен — перетворення комп'ютерних імен на IP-адреси.

**DNS RRL (DNS Response Rate Limiting)**
Обмеження частоти відповідей авторитетних серверів.

**Докінг-станції (Docking stations)**
Периферійні пристрої для портативних обчислювальних пристроїв. Нерідко надають замикне кріплення.

**Домен (Domain)**
Унікальний простір, призначений реєстратором, авторизованим ICANN.

**DoS (Disk Operating System)**
Набір програм для управління загальною роботою комп'ютера.

**DRDoS (Distributed Reflection and Amplification DoS)**
Зловмисник використовує ботнет для атаки на сервери-відбивачі, що виглядають як джерело.

**DTP (Dynamic Trunking Protocol)**
Зловмисник надсилає запит DTP до комутатора для налаштування порту як транкового підключення.

## E

**eCryptfs**
Вбудований пакет шифрування на рівні файлової системи в основних дистрибутивах Linux.

**EFS (Encrypting File System)**
Служба шифрування файлів і папок у Windows.

**Шифрування (Encryption)**
Обробка даних (відкритого тексту) ключовим кодом (шифром) для отримання зашифрованого тексту.

**Корпоративні мережі (Enterprise networks)**
Комп'ютерна мережа для підтримки середньої та великої кількості користувачів у бізнесі.

**eSATA (External Serial AT Attachment)**
Зовнішній фізичний інтерфейс для eSATA-пристроїв.

**ESP (Encapsulating Security Payloads)**
Шифрування та інкапсуляція всього TCP/UDP-датаграму в заголовку ESP.

**Екстрамережа (Extranet)**
Структура інтрамережі з обмеженим доступом для авторизованих зовнішніх користувачів.

## F

**F2FS (Flash-Friendly File System)**
Відкрита файлова система Linux для флеш-сховищ від Samsung.

**Помилкові позитивні (False positive)**
Звіт, що невірно автентифікує особу, надаючи доступ тим, хто не повинен його мати.

**Помилкові негативні (False negative)**
Звіт, що невірно відмовляє особі у доступі.

**Волоконно-оптичний кабель (Fiber optic cabling)**
Пластиковий або скляний кабель для передачі цифрових даних у формі світлових імпульсів.

**Шифрування файлів і папок (File and folder-level encryption)**
Шифрування на рівні окремих файлів і папок.

**FileVault**
Служба шифрування на рівні диска в Apple OS X.

**Мікропрограма (Firmware)**
Постійне програмне забезпечення, запрограмоване в постійну пам'ять.

**Міжмережеві екрани (Firewalls)**
Пристрої (апаратні та/або програмні) для захисту приватної мережі від несанкціонованого доступу.

**Атака Fraggle**
DoS-атака, подібна до Smurf, де зловмисник надсилає підроблені UDP-пакети на широкомовну адресу.

**Кадр (Frame)**
Одиниця цифрової передачі даних.

**FTP-сервер**
Централізований сервер для обміну та зберігання файлів, що використовує протокол FTP.

## G

**Шлюз (Gateway)**
Пристрій, що з'єднує мережу з іншою мережею з іншим протоколом.

**Небажані програми (Grayware)**
Програми, що не класифікуються як шкідливе ПЗ, але містять небажані функції.

**Групові облікові записи (Group accounts)**
Колективний набір користувачів із однаковими критеріями дозволів.

**Гості (Guests)**
Стандартна група з мінімальним доступом.

## H

**Хакер (Hacker)**
Особа, що використовує навички роботи з комп'ютерами для несанкціонованого доступу.

**Посилення захисту (Hardening)**
Процес підвищення безпеки будь-якого пристрою з урахуванням прийнятної зручності використання.

**Апаратний порт (Hardware port)**
Інтерфейс між різними комп'ютерними та мережевими пристроями.

**Апаратний токен (Hardware token)**
Також відомий як токен безпеки — фізичний пристрій для автентифікації.

**Хеш-таблиця (Hash table)**
Таблиця пошуку, що відображає ключі до значень через хеш-функцію.

**Медоловки (Honeypots)**
Приманний сервер, мережевий пристрій або сегмент мережі для відволікання зловмисників від реальної мережі.

**HTTP (Hypertext Transfer Protocol)**
Основний протокол вебзастосунку.

**Гіпервізор (Hypervisor)**
Віртуальна операційна платформа для розміщення інших гостьових ОС.

## I

**IANA (Internet Assigned Numbers Authority)**
Організація, що наглядає за розподілом IP-адрес між ISP.

**ICANN (Internet Corporation for Assigned Names and Numbers)**
Корпорація з управління доменними іменами та числами.

**ICMP**
Протокол керуючих повідомлень Інтернету.

**IDPS (Intrusion Detection and Prevention System)**
Система виявлення та запобігання вторгненням.

**IDS (Intrusion Detection System)**
Система виявлення вторгнень.

**IEEE**
Міжнародна організація з електротехніки та електроніки.

**IETF (Internet Engineering Task Force)**
Основна організація зі стандартів Інтернету.

**IKE (Internet Key Exchange)**
Протокол для встановлення асоціації безпеки в IPsec.

**IoT (Internet of Things)**
Мережа фізичних об'єктів із мережевим підключенням.

**IPsec (Internet Protocol Security)**
Відкритий стандарт для шифрування та автентифікації IP-комунікацій.

**IP-адреса (IP address)**
Унікальний рядок чисел для ідентифікації кожного пристрою, що використовує Інтернет-протокол.

**IP-спуфінг (IP address spoofing)**
Використання підробленої IP-адреси для приховування ідентичності.

**IPv4**
IP-адреси в числовому форматі XXX.YYY.ZZZ.AAA (чотири 8-бітних поля).

**IPv6**
Розширена 128-бітна IP-адресація в шістнадцятковому форматі.

## J

**JavaScript**
Об'єктно-орієнтована мова програмування, часто використовується у браузерах.

**JFFS2 (Journal Flash File System v2)**
Стандартна файлова система Android за замовчуванням.

## K

**Kerberos**
Мережевий протокол автентифікації з довіреним сервером надання квитків (TGS).

**Файли ядра (Kernel files)**
Базові файли ОС для інтерпретації команд від програмного забезпечення.

**Ключ (Key)**
У криптографії — рядок даних для шифрування або розшифрування.

## L

**L2TP (Layer 2 Tunneling Protocol)**
Протокол тунелювання рівня 2 — незашифрований, але може передавати інший протокол шифрування.

**LDAP (Lightweight Directory Access Protocol)**
Популярний корпоративний протокол автентифікації.

**Linux**
Вільно поширювана операційна система з відкритим кодом на основі Unix.

**LAN (Local Area Network)**
Мережа, що існує в відносно обмеженому географічному просторі.

**Локальний міжмережевий екран (Local firewall)**
Міжмережевий екран, що реалізований лише на хості.

**Атака Logjam**
Атака на обмін ключами Диффі-Геллмана, аналогічна FREAK.

**Логічна бомба (Logic bomb)**
Код, прикріплений до легітимної програми, що активується при виконанні певної логічної умови.

## M

**MAC-адреса (MAC address)**
Унікальний ідентифікатор кожного пристрою в мережі.

**MAC-флудинг (MAC flooding)**
Переповнення CAM-таблиці комутатора великою кількістю підроблених MAC-адрес.

**Шкідливе ПЗ (Malware)**
Будь-яка кількість нав'язливих програм зі зловмисною метою.

**MITM-атаки (Man-in-the-middle attacks)**
Зловмисник перехоплює повідомлення між двома або більше жертвами.

**Керовані комутатори (Managed switches)**
Пристрої з програмованими функціями управління.

**MAC (Mandatory Access Control)**
Тип контролю доступу, що встановлює доступ системою.

**Атака маскування (Masquerade attack)**
Зловмисник бере на себе ідентичність іншої системи.

**Меш-мережа (Mesh network)**
Децентралізована топологія, де кожен вузол ретранслює дані.

## N

**Nagios**
Один із найбільш відомих інструментів мережевого моніторингу.

**NIST (National Institute of Standards and Technology)**
Федеральне технологічне агентство, що розробляє рамкові засади кібербезпеки.

**Мережевий міст (Network bridge)**
Апаратний пристрій, що з'єднує кілька мережевих сегментів.

**NAT (Network Address Translation)**
Перетворення IP-адреси однієї мережі на IP-адресу іншої.

**Мережевий адміністратор (Network administrator)**
Відповідає за реалізацію організаційних політик безпеки.

**Мережевий аналізатор (Network analyzer)**
Також аналізатор пакетів — для прослуховування мережевого трафіку.

**NAS (Network-Attached Storage)**
Віртуалізована система зберігання.

**Сегментація мережі (Network segmentation)**
Розподіл мережі на підмережі.

**Мережеві сервери (Network servers)**
Спеціалізовані комп'ютери для ефективної роботи в багатокористувацькому середовищі.

**NX-біт (No eXecution bit)**
Технологія для розподілу ділянок пам'яті за призначенням.

**Дозволи NTFS**
Дозволи New Technology File System для операцій над файлами та папками.

## O

**OSI (Open Systems Interconnection)**
Основна мережева модель, що визначає комунікаційні функції.

**OVAL (Open Vulnerability and Assessment Language)**
Стандарт спільноти безпеки для передачі інформації про вразливості.

**OpenVAS**
Відкрита система сканування вразливостей.

**Операційна система (Operating System, OS)**
Програмне забезпечення для управління та координації роботи комп'ютерної системи.

**Накладні мережі (Overlay networks)**
Мережа, побудована поверх іншої фізичної мережі.

## P

**Пакет (Packet)**
Одиниця даних, що переноситься мережею.

**Аналізатор пакетів (Packet analyzer)**
Також аналізатор мережі — для аналізу мережевого трафіку.

**Фільтрація пакетів (Packet filtering)**
Пропускання або блокування мережевих пакетів на основі адрес, портів або протоколів.

**PAKE (Password-Authenticated Key Agreement)**
Зашифрована автентифікація з використанням спільних ключів.

**PAN (Personal Area Network)**
Мережа особистих пристроїв.

**PAP (Password Authentication Protocol)**
Стандартна схема ім'я користувача та пароля.

**PAT (Port Address Translation)**
Відображення кількох внутрішніх IP на єдину зовнішню IP-адресу з доданим номером порту.

**Пасивний інфрачервоний детектор (PIR detector)**
Датчик для виявлення змін інфрачервоної енергії.

**PCI-DSS (Payment Card Industry Data Security Standard)**
Стандарт безпеки платіжних карток.

**PGP (Pretty Good Privacy)**
Широко використовуваний інструмент шифрування Unix.

**Фішинг (Phishing)**
Спроба отримати конфіденційну інформацію, маскуючись під надійну організацію.

**Пікомережа (Piconet)**
Мала Bluetooth-мережа до восьми пристроїв.

**PING**
Утиліта для оцінки досяжності IP-хоста.

**PKI (Public Key Infrastructure)**
Інфраструктура для розповсюдження відкритих ключів і сертифікатів.

**PKE (Public Key Encryption)**
Шифрування з двома ключами: відкритим (для шифрування) та приватним (для розшифрування).

**Plug and Play (PnP)**
Автоматичне налаштування пристроїв.

**POODLE**
*Padding Oracle on Downgraded Legacy Encryption* — атака на SSL v3.0.

**Порт (Port)**
Логічна точка підключення для мережевих сервісів.

**POSIX**
Набір стандартів сумісності для Unix та Unix-подібних ОС.

**Проксі-фільтруючі міжмережеві екрани (Proxy-filtering firewalls)**
Сервери для фільтрації небажаних пакетів шляхом розбирання, оцінки та повторного збирання.

**Проксі-сервер (Proxy server)**
Дозволяє клієнтам здійснювати непрямі мережеві підключення.

## Q

**QoS (Quality of Service)**
Пріоритизація та диференційована обробка мережевого трафіку.

## R

**Атака на умову гонки (Race condition attack)**
Зловмисник використовує тимчасовий розрив між послідовними подіями для вставки шкідливого коду.

**RADIUS (Remote Authentication Dial-In User Service)**
Протокол мережевої автентифікації та обліку.

**Ransomware**
Шифрує файли та вимагає оплату за відновлення доступу.

**RBAC (Role-Based Access Control)**
Рольовий контроль доступу — базується на посадових ролях.

**Сервери дистанційного доступу (RAS, Remote Access System)**
Дозволяють клієнтам підключатися з віддаленого місця.

**Відтворення сеансу (Session Replay attack)**
Зловмисник записує, маніпулює та повторно вводить IP-пакети.

**Перенаправлення (Rerouting attack)**
Зловмисник отримує доступ до таблиць маршрутизації та переналаштовує маршрутизатор.

**RFC (Request for Comments)**
Примітки, що подаються до IETF для стандартизації.

**Rootkit**
ПЗ для отримання адміністративного контролю з приховуванням своєї присутності.

**Флудинг маршрутизаторів (Router flood attacks)**
Атаки на споживання ресурсів маршрутизатора.

**Маршрутизатори (Routers)**
Пристрої мережевого підключення, що пересилають інформацію між мережевими сегментами.

**Маршрутизація (Routing)**
Вибір найкращого шляху для передачі даних.

## S

**SAN (Storage Area Network)**
Мережева система зберігання.

**SCAP (Security Content Automation Protocol)**
Метод використання відкритих стандартів для оцінки вразливостей.

**Secure Shell (SSH)**
Зашифрований мережевий протокол для захищених клієнт-серверних підключень.

**SSL (Secure Sockets Layer)**
Протокол управління автентифікацією та комунікаціями між клієнтами та серверами.

**TLS (Transport Layer Security)**
Наступник SSL.

**Сегментація мережі (Network segmentation)**
Розподіл мережі на підмережі — також відоме як зонування.

**Сесійний cookie (Session cookie)**
Зберігається у тимчасовій пам'яті; видаляється при закритті браузера.

**Викрадення сесії (Session hijacking)**
Отримання сесійного cookie та несанкціонований доступ.

**Підпис (Signature analysis)**
Порівняння трафіку з базою даних підписів шкідливих загроз.

**Смарт-картка (Smart card)**
Фізичний пристрій із інформацією для автентифікації.

**Атака Smurf**
DDoS-атака ICMP-відповідями з підробленими IP-адресами жертви.

**Snort**
Відкрита мережева IDS для аналізу трафіку в реальному часі.

**Соціальна інженерія (Social engineering)**
Психологічна маніпуляція для отримання інформації.

**Програмне забезпечення безпеки (Software-defined networking, SDN)**
Підхід, що аналізує підключення між вузлами та фільтрує на основі політики.

**SQL-ін'єкція (SQL injection)**
Введення SQL-команд у поля форм для несанкціонованого доступу до бази даних.

**Адресний фішинг (Spear phishing)**
Фішинг, що спрямований на конкретну організацію або особу.

**Шпигунське ПЗ (Spyware)**
Програмне забезпечення, що відстежує системну активність і збирає інформацію.

**Підмережа (Subnet)**
Сегментована мережа, що є частиною більшої мережі.

**Суперкукіs (Super cookies)**
Сторонні cookie, що важче видалити; нерідко використовують HTML5 або Flash.

**Комутатори (Switches)**
Пристрої мережевого підключення Рівня 2.

**SYN-флуд (SYN flood)**
Форма DoS-атаки, де зловмисник використовує тристороннє рукостискання TCP.

## T

**TCP/IP**
Набір протоколів — основа Інтернет-моделі.

**Атака сльозою (Teardrop attack)**
DoS-атака з фрагментованими пакетами.

**Telnet**
Протокол клієнт-сервер для підключення до віддаленого Telnet-сервера.

**Порогові системи виявлення аномалій (Threshold-based anomaly detection)**
Відстеження кількості конкретних подій за час; попередження при перевищенні порогу.

**TPM (Trusted Platform Module)**
Вбудований мікрочіп для зберігання криптографічної інформації.

**Tarpitting**
Відповідь на підключення з дуже маленьким розміром вікна TCP для сповільнення зловмисників.

**Trojans**
Шкідливе ПЗ, що маскується під легітимні застосунки.

**Тунелювання (Tunneling)**
Захищене підключення дистанційних користувачів до внутрішніх ресурсів.

**Двофакторна автентифікація (Two-factor authentication)**
Вимога двох різних факторів для авторизації.

## U

**UDP flood attack**
DoS-атака з UDP-пакетами.

**UTM (Unified Threat Management)**
Апарати безпеки з комплексними функціями захисту в одному продукті.

**Unicode**
Unicast — комунікація між одним відправником і одним одержувачем.

**UPnP (Universal Plug and Play)**
Мережева архітектура з нульовою конфігурацією для сумісності між різними пристроями.

**USB (Universal Serial Bus)**
Найпоширеніший апаратний порт у сучасних ПК.

**Unix**
Модульна, багатозадачна, багатокористувацька ОС.

## V

**VBScript**
Мова сценаріїв для Microsoft IE.

**VLAN (Virtual LAN)**
Програмно налаштована мережа для логічного об'єднання пристроїв.

**VPN (Virtual Private Network)**
Дозволяє дистанційним користувачам підключатися до приватної мережі через публічну.

**Віруси (Viruses)**
Шкідливі програми для реплікації та поширення.

**Вразливість (Vulnerability)**
Будь-яка слабкість, що може дозволити зловмиснику доступ до мережевих активів.

**Сканери вразливостей (Vulnerability scanners)**
Інструменти на основі баз даних для пошуку відомих вразливостей.

## W

**Веббраузер (Web browser)**
Програмний застосунок для отримання та перегляду ресурсів в Інтернеті.

**Вебпроксі (Web proxy)**
Комп'ютер або сервер, що функціонує від імені хоста або мережі хостів.

**Вебсервер (Web server)**
Сервер для розміщення вебсторінок.

**WEP (Wired Equivalent Privacy)**
Протокол безпеки для WLAN із 128-бітним шифруванням.

**Бездротова точка доступу (Wireless Access Point, WAP)**
Пристрій підключення для бездротових клієнтів.

**Wireshark**
Зрілий, безкоштовний та міжплатформний аналізатор мережевих протоколів.

**WiMAX**
Стандарт бездротового широкосмугового доступу для великих географічних площ.

**Wi-Fi**
Бездротова мережева технологія для підключення пристроїв до LAN.

**WPA (Wi-Fi Protected Access)**
Більш надійний стандарт бездротового шифрування, ніж WEP.

**Хробаки (Worms)**
Шкідливі програми, що поширюються через мережеве підключення.

## Z

**Вразливість нульового дня (Zero day vulnerability)**
Вразливість, невідома постачальнику продукту, без наявного патча.

**Зомбі (Zombies)**
Заражені комп'ютери під дистанційним контролем зловмисника.

**ZigBee**
Бездротовий PAN-протокол із сітчастою топологією, дальністю 10 метрів.

**Z-Wave**
Стандарт бездротового зв'язку для систем домашньої автоматизації.
