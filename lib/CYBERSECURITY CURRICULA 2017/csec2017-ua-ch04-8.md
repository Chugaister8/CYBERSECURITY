# 4.8 Галузь знань: Суспільна безпека

Галузь знань «Суспільна безпека» зосереджена на аспектах кібербезпеки, що широко впливають на суспільство в цілому — як позитивно, так і негативно. Кіберзлочинність, право, етика, політика, конфіденційність та їхній взаємозв'язок є ключовими концепціями цієї галузі знань. Загроза кіберзлочинності для глобального суспільства є надзвичайно серйозною і зростає. Закони, етика та політики є життєво важливими для захисту корпоративних та державних таємниць і активів, а також для захисту особистої конфіденційності та ідентичності.

## 4.8.1 Одиниці знань та теми

Наведена нижче таблиця перелічує основні елементи, одиниці знань та теми галузі знань «Суспільна безпека».

---

**СУСПІЛЬНА БЕЗПЕКА**

**Основні елементи:**
- Кіберзлочинність,
- Кіберправо,
- Кіберетика,
- Кіберполітика,
- Конфіденційність.

---

|---|---|---|
| **Кіберзлочинність** | | Ця одиниця знань має на меті надати студентам розуміння масштабів, вартості та правового середовища, пов'язаних із крадіжкою інтелектуальної власності в кіберпросторі. Це включає як національне, так і міжнародне середовища. | | Поведінка кіберзлочинців | Поведінка, що атакує комп'ютерний пристрій або комп'ютерну інфраструктуру окремих осіб/компаній для здійснення зловмисних дій, таких як поширення вірусів, крадіжка даних та крадіжка ідентичності. |

| | Економіка кіберзлочинності | Ризики кіберзлочинності є надто низькими, тоді як вигоди є надто високими; використання (невідстежуваних) криптовалют у вчиненні кіберзлочинів в Інтернеті та в Даркнеті (bitcoin). |
| **Кіберправо** *[Див. також KA «Організаційна безпека» щодо пов'язаного змісту, с. 51.]* | | Ця одиниця знань має на меті надати студентам широке розуміння поточного правового середовища стосовно кіберпростору. Це включає як внутрішнє, так і міжнародне право, а також застосування юрисдикційних меж у кіберюридичних справах. Студенти повинні добре розуміти чинне застосовне законодавство та мати міцну основу у сфері формування цих правових інструментів. |
| | Конституційні основи кіберправа | Ця тема включає: виконавчу владу; законодавчу владу; Першу поправку; Четверту поправку; Десяту поправку (до Конституції США). |
| | Інтелектуальна власність, пов'язана з кібербезпекою *[Див. також KA «Безпека даних», с. 16, KA «Безпека людини», с. 44, та KA «Організаційна безпека», с. 51, щодо пов'язаного змісту.]* | Ця тема охоплює: масштаби, вартість та правове середовище, пов'язані з крадіжкою інтелектуальної власності в кіберпросторі; конкретний зміст визначатиметься країною, на яку зроблено акцент. У США охоплюється Розділ 1201 Закону про авторське право в цифровому тисячолітті (DMCA); антиобходові заходи — DMCA 1201. |
| | Закони про конфіденційність *[Див. також KA «Безпека даних», с. 16, KA «Безпека людини», с. 44, та KA «Організаційна безпека», с. 51, щодо пов'язаного змісту.]* | Ця тема включає: закони, що регулюють конфіденційність в Інтернеті; закони, що регулюють конфіденційність у соціальних мережах; закони про електронне спостереження, такі як Wiretap Act, Stored Communications Act та Pen Register Act. |
| | Законодавство у сфері безпеки даних | Ця тема включає: Розділ 5 Федеральної торгової комісії США; закони штатів щодо безпеки даних; закони штатів про сповіщення про порушення безпеки даних; Закон про мобільність і підзвітність медичного страхування (HIPAA); Закон Грема-Ліча-Блілі (GLBA); обмін інформацією через US-CERT, Закон про кібербезпеку 2015 року. |
| | Закони про комп'ютерне хакерство | Ця тема охоплює: федеральні закони США про комп'ютерні злочини, такі як Закон про шахрайство та зловживання комп'ютерами (більшість правопорушень, пов'язаних із комп'ютерним хакерством, переслідуються в США відповідно до Закону про шахрайство та зловживання комп'ютерами); міжнародну правову базу та співробітництво, необхідне для переслідування хакерів з-за кордону. |
| | Цифрові докази | Ця тема включає: криміналістично коректний збір цифрових доказів; збереження ланцюжка зберігання. |
| | Цифрові контракти | Ця тема включає: розмежування між угодами browse-wrap, click-wrap та shrink-wrap; Закон про електронні підписи в глобальній та міжнародній торгівлі (ESIGN) 2000 року (цифрові контракти та електронні підписи є такими ж юридично дійсними та обов'язковими, як і традиційні паперові контракти, підписані чорнилом). |
| | Багатонаціональні конвенції (угоди) | Ця тема охоплює юрисдикційні обмеження багатонаціональних угод. Приклади: Будапештська конвенція про кіберзлочинність та Угода G-7 про кібербезпеку фінансових установ. |
| | Транскордонні закони про конфіденційність та безпеку даних | Вимоги Загального регламенту про захист даних (GDPR). Угода Privacy Shield між країнами, такими як Сполучені Штати та Сполучене Королівство, що дозволяє передавання персональних даних. |
| **Кіберетика** *[Див. також KA «Організаційна безпека», с. 51, та KA «Безпека програмного забезпечення», с. 23, щодо пов'язаного змісту.]* | | Ця одиниця знань має на меті надати студентам основу для розуміння та застосування моделей морального міркування при розгляді поточних і нових етичних дилем на індивідуальному та груповому (професійному) рівнях. Вона також сприяє усвідомленню студентами дискусій про те, чи є етика в обчислюваннях унікальною проблемою чи частиною ширшого явища, та допомагає студентам осмислити вплив культури та правової бази їхньої країни на розуміння та реалізацію етики в їхньому суспільстві. |
| | Визначення етики | У межах цієї теми: порівняти та зіставити основні етичні позиції, включаючи етику чеснот, утилітарну етику та деонтологічну етику; застосовувати три різні етичні позиції при осмисленні етичних наслідків конкретної проблеми або дії. |
| | Професійна етика та кодекси поведінки | Ця тема охоплює: провідні професійні товариства, такі як ACM, IEEE-CS, AIS та (ISC)²; професійну відповідальність; етичну відповідальність стосовно спостереження. |
| | Етика та рівність/різноманіття | У межах цієї теми: описати способи, якими алгоритми прийняття рішень можуть надмірно або недостатньо представляти більшість і меншість у суспільстві; аналізувати способи, якими алгоритми можуть неявно включати соціальні, гендерні та класові упередження. |
| | Етика та право | У межах цієї теми: розуміти, що етичні практики та правові норми не завжди точно збігаються; етичні практики можуть розглядатися як універсальні, тоді як закони можуть бути специфічними для країни або регіону (наприклад, Європейський Союз); закони можуть змінюватися, але етичні цінності можна охарактеризувати як незмінні. |
| | Етика автономії/роботів | У межах цієї теми: визначити автономне прийняття рішень; визначити штучний інтелект та описати етичні дилеми, що виникають при використанні або застосуванні штучного інтелекту (AI); описати законодавчі досягнення, що визначили правосуб'єктність та цифрову правосуб'єктність; описати конфлікт, що виникає між правовими поняттями відповідальності та використанням безпілотних або автономних програм прийняття рішень. |
| | Етика та конфлікт | Ця тема включає: принципи справедливої війни стосовно кіберпростору в контексті ініціювання конфлікту, поведінки в конфлікті, припинення конфлікту/ситуації після конфлікту; етичні проблеми, що виникають при проведенні кібершпигунства; порушення норм та правил стосовно кібертероризму. |
| | Етичне хакерство | Ця тема включає: етичне тестування на проникнення на противагу неетичному хакерству; принципи та умови етичного хакерства; розмежування між хакерством-дрібним хуліганством, хактивізмом, кримінальним хакерством та актами війни. |
| | Етичні фреймворки та нормативні теорії | Поширені етичні фреймворки та нормативні теорії, пов'язані з кібербезпекою, з індивідуальної та суспільної точки зору. |
| **Кіберполітика** *[Див. також KA «Організаційна безпека» щодо пов'язаного змісту, с. 51.]* | | Одиниця знань «Кіберполітика» призначена допомогти студентам розуміти та аналізувати кіберпроблеми стосовно національних інтересів загалом, а також національної (та національної безпеки) політики зокрема. Від студентів очікується розуміння питань, пов'язаних із використанням кіберпростору як інструменту війни, та розмежування між таким використанням кіберпростору та можливістю виникнення кібервійни як такої. Студентам надається можливість осмислити питання про те, як використання кіберпростору може сигналізуватися іншим країнам, а також виклики, пов'язані із стримуванням. Від студентів також очікується розуміння історичних тенденцій, що зробили кіберпростір важливим для національної політики, та розвиток архітектури національної кіберполітики. Студенти повинні продемонструвати оригінальне мислення щодо того, як кіберпростір впливає на національні інтереси, включаючи економічні, та наслідки для національної політики, що випливають із кіберпростору. |
| | Міжнародна кіберполітика | Ця тема включає: виклики міжнародної кіберполітики; Закон про нагляд за міжнародною кіберполітикою 2015 року; стратегію міжнародної кіберполітики Державного департаменту. |
| | Федеральна кіберполітика США | Ця тема включає: Закон про модернізацію федеральної безпеки інформації — оновлення політик і рекомендацій федерального уряду з кібербезпеки; зв'язок із критичною інфраструктурою країни; управління ризиками на національному рівні. |
| | Глобальний вплив | Ця тема охоплює: вплив кібербезпеки на міжнародну систему загалом та на міжнародну безпеку зокрема; те, як кіберпростір став і продовжить ставати інструментом влади, та те, як ця влада може змінити баланс сил між сильнішими та слабшими країнами; глобальне управління кіберпростором, а також розгляд можливостей розвитку нормативної поведінки стосовно використання кіберпростору; вплив кіберпростору на глобальну економіку. |
| | Кіберполітика та національна безпека | Ця тема охоплює: те, як країна визначає свою кіберполітику, доктрину та відповідальність за виконання, включаючи національну кіберполітику, архітектуру, сигнали та наративи, а також примус та демонстрацію сили; кіберповідомлення країни; те, як вона сигналізує про свої наміри, щоб привернути увагу та отримати співпрацю інших країн. |
| | Національні економічні наслідки кібербезпеки | Ця тема охоплює: вартість кібербезпеки для країни; збитки та вигоди кібербезпеки для країни; інвестиції для захисту країни від кіберзагроз та кібератак. |
| | Нові суміжні сфери дипломатії | Ця тема включає: «делікатний танець» кібердипломатії; аспекти кібербезпеки, що стали частиною відносин між країнами, включаючи таємний збір інформації поряд із дипломатичною практикою, а також таємне застосування кіберсили в кіберпросторі та фізичному просторі. |
| **Конфіденційність** *[Див. також KA «Безпека людини», с. 44, KA «Організаційна безпека», с. 51, та KA «Безпека даних», с. 16, щодо пов'язаного змісту.]* | | Ця одиниця знань призначена надати студентам розуміння конфіденційності та пов'язаних із нею викликів. Від студентів очікується розуміння компромісів між поширенням та захистом чутливої інформації, а також того, як внутрішні та міжнародні права на конфіденційність впливають на відповідальність компанії за збір, зберігання та обробку персональних даних. Студенти отримають розуміння технологій підвищення конфіденційності та застосування засобів безпеки, що може включати концепції належного використання, а також захисту інформації. |
| | Визначення конфіденційності | У межах цієї теми: застосовувати операційні визначення конфіденційності; визначати різні цілі конфіденційності, наприклад конфіденційність комунікацій та конфіденційність метаданих; визначати компроміси конфіденційності — підвищення конфіденційності може нести ризики (наприклад, використання Tor може зробити когось об'єктом посиленої державної уваги в деяких частинах світу). |
| | Права на конфіденційність | У межах цієї теми: описати умови інформованої згоди стосовно збору та поширення персональних даних; визнавати національні права на конфіденційність у контексті існування прав на конфіденційність; продемонструвати обізнаність із дискусією про загальне право людини на конфіденційність. |
| | Захист конфіденційності | У межах цієї теми: перелічити кроки кібергігієни для захисту особистої конфіденційності; перелічити технології підвищення конфіденційності та їхнє використання, а також властивості, які вони забезпечують і не забезпечують (тобто Tor, шифрування); описати умови для етичного та законного використання технологій підвищення конфіденційності; описати кроки проведення оцінювання впливу на конфіденційність; описати роль довіреного управителя даних; описати законодавство, пов'язане з практиками локалізації даних; продемонструвати розуміння різниці між правами на конфіденційність та можливостями підвищення конфіденційності — операціоналізація конфіденційності; обговорити динамічний вплив метаданих та великих даних на конфіденційність. |
| | Норми та ставлення до конфіденційності | Ця тема включає: теорію та моделі обрахунку конфіденційності; культурні відмінності в існуванні норм та меж конфіденційності. |
| | Порушення конфіденційності | Ця тема охоплює роль корпорацій у захисті даних та реагуванні на обставини, коли конфіденційність даних порушується. |
| | Конфіденційність у суспільствах | Ця тема включає: права на конфіденційність та загрози конфіденційності, пов'язані з публічними особами; диференційоване спостереження та його ризики; виклики для розумних міст; матрицю шкоди для кіберспостереження. |

## 4.8.2 Основні елементи та результати навчання

Від студентів вимагається демонструвати компетентність у кожному з основних понять через досягнення результатів навчання. Як правило, результати навчання лежать у межах рівнів розуміння та застосування за переглянутою таксономією Блума (http://ccecc.acm.org/assessment/blooms).

| Основні елементи | Результати навчання |
|---|---|
| Кіберзлочинність | Обговорити різні мотиви кіберзлочинної поведінки. |
| | Підсумувати терористичні дії в кіберпросторі, спрямовані на поширення суспільного страху та невизначеності. |
| | Описати методи розслідування як внутрішніх, так і міжнародних злочинів. |
| | Пояснити, чому збереження ланцюжка цифрових доказів є необхідним при судовому переслідуванні кіберзлочинів. |
| Кіберправо | Описати конституційні основи кіберправа. |
| | Описати міжнародне законодавство у сфері безпеки даних та комп'ютерного хакерства. |
| | Тлумачити закони про інтелектуальну власність, пов'язані з безпекою. |
| | Підсумувати закони, що регулюють онлайн-конфіденційність. |
| Кіберетика | Розмежувати етику чеснот, утилітарну етику та деонтологічну етику. |
| | Перефразувати професійну етику та кодекси поведінки провідних професійних товариств, таких як ACM, IEEE-CS, AIS та (ISC)². |
| | Описати способи, якими алгоритми прийняття рішень можуть надмірно або недостатньо представляти більшість і меншість у суспільстві. |
| Кіберполітика | Описати основні позиції міжнародної публічної політики та їхній вплив на організації та осіб. |
| | Підсумувати національну публічну кіберполітику конкретних країн стосовно захисту чутливої інформації та критичної інфраструктури. |
| | Пояснити глобальний вплив кібербезпеки на культуру, включаючи такі галузі, як економіка, соціальні питання, політика та законодавство. |
| Конфіденційність | Описати концепцію конфіденційності, включаючи суспільне визначення того, що є особисто приватною інформацією, та компроміси між особистою конфіденційністю та безпекою. |
| | Підсумувати компроміс між правами особи на конфіденційність та потребами суспільства. |
| | Описати поширені практики та технології, що використовуються для захисту особистої конфіденційності. |
